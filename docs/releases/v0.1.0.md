# Discrete Runtime v0.1.0

Discrete Runtime 的首个公开版本。

这是一个与具体玩法和 UI 解耦的离散阶段运行时，提供 Godot 4.6 / GDScript
参考实现，可用于回合制战斗、卡牌和其他离散规则系统。

## 安装

下载 `discrete-runtime-godot-0.1.0.zip`，将其中的：

```text
addons/discrete_runtime/
```

复制或合并到 Godot 项目的 `res://addons/` 下。Runtime 使用 `class_name` 提供类型，
不需要 Autoload，也不是 EditorPlugin。

## 本版内容

- 确定性的阶段内调度与 Effect/Fact 因果链。
- 同步应用与异步 Operation 边界。
- `IMMEDIATE` 和 `AFTER_SETTLEMENT` 两种 Signal 结算出口。
- Runtime 生命周期及失败保护。
- 最小示例与 Runtime 契约测试。

当前验证基线为 Godot 4.6.3。

## 版本状态

`v0.1.0` 已具备稳定、可测试的核心语义，但公共 API 在 `1.0.0`
之前仍可能根据实际使用反馈调整。
