Runtime 发布目录 `addons/discrete_runtime/` 不包含第三方代码。

源码仓库为运行开发测试固定保存了 GUT 9.6.1：

- 项目：GUT - Godot Unit Test
- 作者：Tom “Butch” Wesley 及贡献者
- 许可证：MIT
- 位置：`addons/gut/`
- 完整许可证：`addons/gut/LICENSE.md`

GUT 自带字体文件继续遵循 `addons/gut/fonts/OFL.txt` 中的 SIL Open Font License 1.1。

`addons/gut/` 及其字体只属于开发测试基础设施，不会进入 Discrete Runtime 的用户安装包。
